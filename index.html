ARCHITECTURE

Designed for degradation. Functions when infrastructure fails.

Local-First

Data lives on devices. Sync is peer-to-peer. Central servers are optional relays, not authorities.

Identity-Minimal

No global usernames. No follower graphs. Cryptographic keys represent membership. Keys rotate.

Forkable

Cells can fork. Members can leave with their data. No update is mandatory. No kill switch exists.

Composition Engine

Constraint satisfaction + graph optimization. Outputs viable cells ranked by resilience, not popularity.

COMPOSITION ENGINE

Group formation as constraint-satisfaction, not matchmaking.

INPUTS
  • Skills graph
  • Dependency graph
  • Geography
  • Redundancy thresholds
OPTIMIZES FOR
  • Complementary skills
  • No single point of failure
  • Geographic proximity
  • Minimal shared dependencies
Output: "This group can function if two members disappear."
